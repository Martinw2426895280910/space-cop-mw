# 🚀 ASTRO COP — Alien Invasion

Juego móvil de astronauta vs aliens. Canvas HTML5 puro, sin dependencias externas.

## 🎮 Gameplay
- Astronauta dibujado con traje blanco/naranja, casco con visor reflectante y jetpack
- Aliens insectoides biomecánicos con tentáculos luminosos y ojos brillantes
- Alien Queen Boss desde nivel 3 con barra de vida épica
- Portal de escape para subir de sector

## 🌌 5 Escenarios únicos
1. Nave Espacial Abandonada (luces parpadeantes)
2. Cinturón de Asteroides (rocas flotantes animadas)
3. Nebulosa Colorida (púrpura y cian espectacular)
4. Superficie Lunar (cráteres y estructuras alienígenas)
5. Nave Madre Alienígena (bioluminiscente y amenazante)

## ⚡ Power-ups
- 🛡 Escudo de Energía
- 🔥 Overcharge (disparo triple)
- 🚀 Jetpack Boost (velocidad extrema)
- ⏱ Time Slow (ralentiza enemies)
- ☢ Nuke (explosión masiva)
- 💊 Medkit (+1 vida)

## 📱 Controles móvil
- Joystick táctil (izquierda)
- Botón FIRE grande (derecha)
- Botón NOVA — onda de choque repulsiva
